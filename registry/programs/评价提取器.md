# 评价提取器

## 适用平台
淘宝、1688

## 淘宝方式
1. 点击 "查看全部评价"（Playwright 原生 click）
2. pg.mouse.wheel(0,800) 逐段滚动
3. 停止条件：连续8步无新Comment元素 或 250+元素

## 1688方式
1. 点击 .evaluation-more .ant-btn → 打开Drawer
2. 读 ant-pagination-item 总页数
3. 逐页点击翻页
4. 关闭 Drawer: pg.keyboard.press('Escape')

## 日期格式
- 预览区: 2026-06-08
- 完整区: 2026年6月8日

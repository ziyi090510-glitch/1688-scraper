# InstHunter 踩过的坑

1. **pydantic-settings .env路径陷阱**：nohup/systemd启动后CWD不是代码目录→配置全读不到
   - 修复：config.py中用__file__动态计算绝对路径

2. **占位符覆盖API Key**：前端保存时`********`覆盖了真实Key
   - 修复：后端判断`if value != "********"`

3. **时区不统一**：多处使用UTC导致数据错乱
   - 修复：6个文件14处全部改为北京时间

4. **IG代理API超时**：海外VPS网络不稳定
   - 修复：多源降级策略

5. **跨服务器部署路径不一致**：20服务器→152服务器路径适配
   - 修复：配置项集中管理，路径相对化

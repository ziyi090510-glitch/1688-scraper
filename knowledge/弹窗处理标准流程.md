# 弹窗处理标准流程

## 检测方式（5维）
1. 文本关键词：操作频繁、验证码、拖动滑块、人机验证、请稍后再试
2. DOM结构：nc_*滑块、captcha/verify元素、验证码输入框
3. URL重定向：verify/captcha/risk/login.taobao.com
4. 页面标题：验证/安全/异常/登录
5. baxia遮罩：computedStyle高z-index固定定位（非inline style）

## 检测频率
- 每次pg.goto() 前 + 后
- 评论滚动前 + 后
- 每个商品处理前

## 处理规则
- 弹窗出现 → 立即停止所有操作
- 发送Bark通知
- 等待用户手动处理
- 绝不自动关闭弹窗（bot行为特征）
- 等待期间每10秒检查一次
- 解除后等待8-15秒再继续

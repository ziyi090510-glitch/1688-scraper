# 1688 爬取注意事项

## 搜索
- 必须用 GBK 编码关键词
- URL: `https://s.1688.com/selloffer/offer_search.htm?keywords={gbk_encoded}`
- offerId 从 `offerIds=` 参数提取

## 评价提取
1. 点击 `.evaluation-more .ant-btn` → 打开 Ant Design Drawer
2. 读 `.ant-pagination-item` 最大数字 = 总页数
3. 逐页点击 `ant-pagination-item-N`
4. 从 `.ant-drawer-body` textContent 提取
5. 评论格式: `用户名 X天前 已购:N个 SKU... 内容`
6. 关闭 Drawer: `pg.keyboard.press('Escape')`

## 刷单过滤
- 用户名: t + *** + 数字 → 刷单
- 内容 <8 字 → 无效
- 中文 <4 字 → 无效
- 同一商品出现 ≥3 次相同 6-8 字短句 → 刷单

## 速度控制
- 操作间隔: 8-14秒
- 搜索间隔: 4-8秒
